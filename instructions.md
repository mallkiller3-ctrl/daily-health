
# ZenHealth Pro 모바일 사용 가이드

본 애플리케이션은 모바일 웹 환경에서 최적의 경험을 제공하도록 설계되었습니다. 핸드폰에서 앱처럼 편리하게 사용하시려면 아래 단계를 따라주세요.

## 📱 홈 화면에 추가하기 (PWA 스타일)

### iPhone (iOS Safari)
1. Safari 브라우저에서 이 페이지를 엽니다.
2. 하단 중앙의 **공유 버튼** (정사각형에 위 화살표 아이콘)을 누릅니다.
3. 메뉴를 아래로 내려 **'홈 화면에 추가'**를 선택합니다.
4. 이름을 설정하고 '추가'를 누르면 바탕화면에 아이콘이 생성됩니다.

### Android (Chrome)
1. Chrome 브라우저에서 이 페이지를 엽니다.
2. 우측 상단의 **점 3개 아이콘**을 누릅니다.
3. **'홈 화면에 추가'** 또는 **'앱 설치'**를 선택합니다.
4. 안내에 따라 설치를 완료하면 바탕화면에서 바로 접속 가능합니다.

## 💡 주요 기능 활용 팁
- **AI 식단 분석**: 음식 이름과 대략적인 양을 입력하면 Gemini AI가 칼로리를 자동으로 계산해줍니다.
- **마운자로 관리**: 프로필에서 활성화하면 대시보드에서 투여 여부를 체크하고 체중 변화 추이를 더 정밀하게 모니터링할 수 있습니다.
- **피부 코칭**: 아침/저녁 루틴을 체크하면 기록된 데이터를 바탕으로 연령대에 맞는 피부 관리 팁을 제공합니다.
- **데이터 저장**: 모든 데이터는 브라우저의 로컬 스토리지에 안전하게 저장됩니다.
